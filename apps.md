## Системные
### yay
**yay** (Yet another Yogurt) — это инструмент для управления пакетами в дистрибутиве Arch Linux, выполняет роль помощника для автоматизации работы с пользовательским репозиторием Arch User Repository (AUR).

1. Для сборки пакета понадобятся инструменты разработчика и Git:
``` bash
sudo pacman -S --needed base-devel git
```
2. Скачать исходный код yay из пользовательского репозитория:
``` bash
git clone https://aur.archlinux.org/yay.git
```
3. Запустить процесс сборки:
``` bash
cd yay
makepkg -si
```
4. Проверить версию:
``` bash
yay --version
```

## Прикладные
### TeamSpeak
**Способ 1:**
``` bash
yay -S teamspeak
```
**Способ 2:**
1. Скачать архив с [официального сайта](https://teamspeak.com/en/downloads/).
2. Разархивировать его:
``` bash
tar -xvzf teamspeak-client.tar.gz
```
3. Установить пакеты:
``` bash
sudo pacman -S --needed nss libxss libxkbcommon alsa-lib libpulse libvdpau vulkan-driver libnotify
```
### Code OSS (VS Code)
``` bash
pacman -S code
```

**Расширения:**
- **Markdown All in One** by yzhang
  - Инструмент для работы с Markdown-файлами, который автоматически управляет нумерацией списков, генерирует кликабельное оглавление и добавляет горячие клавиши для форматирования текста.
- **vscode-pdf** by tomokl1207
  - Встроенный просмотрщик PDF-документов.
- **shell-format** by foxundermoon
  - Легкий автоматический форматировщик для Bash и Shell-скриптов.
- **ShellCheck** by timonwong
  - Статический анализатор кода для поиска багов в скриптах, подсвечивающий скрытые синтаксические ошибки, забытые кавычки и потенциально опасные команды.
- **Todo Tree** by Gruntfuggly
  - Инструмент для автоматического сбора задач и чекбоксов по всему проекту, который сканирует заметки и код, находит теги вроде TODO или [ ] и выстраивает их в красивое интерактивное дерево на боковой панели.
- **Foam** by foam
  - Превращает редактор в аналог Obsidian за счет связывания заметок между собой и построения красивого интерактивного графа взаимосвязей, помогая визуализировать структуру базы знаний.
