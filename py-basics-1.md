## Введение
| Тип | Принцип работы | Примеры |
|-----|----------------|---------|
| **Компилируемые** | Проверка синтаксиса → перевод в машинный код | C, C++, Pascal |
| **Интерпретируемые** | Анализ и покомандное выполнение кода | Python, PHP, Ruby |
| **Смешанные (Байт-код)** | Компиляция в промежуточный код → перевод в машинный | C#, Java, Python |

- **IDE** – Integrated Development Environment.
- **Присваивание**: оператор `=` (не путать с условным оператором `==`).
- **Комментарии**: `#` (однострочные), `'''...'''` или `"""..."""` (многострочные/строки).

```python
description = '''Это многострочный текст.
Он сохраняет все переносы строк.''' ' А это продолжение той же строки.'

print(description)

# Присваивание нескольких переменных в одной строке через точку с запятой
status = "active"; user_id = 101
```

## Строки (String)
### Основные операции
- **Конкатенация**: сложение строк. **Не является коммутативной** (результат зависит от порядка).
- **Изменять отдельные символы нельзя**, можно только создавать новые строки или изменять элементы списков.

### Форматирование
```python
# .format()
# Использование индексов аргументов {0}, {1}
print('Пользователь {0} набрал {1} очков.'.format(user, score))
# Использование пустых скобок {} (автоматическая нумерация)
print('Результат игрока {} равен {}'.format(user, score))

# f-strings (современный аналог)
print(f'Hello, {first_name} {last_name}. You are {age}.')
```

### Методы изменения регистра
| Метод | Описание | Пример |
|-------|----------|--------|
| `capitalize()` | 1 символ верхний, остальные нижний | `'foO BaR'` → `'Foo bar'` |
| `swapcase()` | Меняет регистр на противоположный | `'FOO Bar'` → `'foo bAR'` |
| `title()` | 1 символ каждого слова верхний | `'the sun'` → `'The Sun'` |
| `lower()` / `upper()` | Все строчные / все заглавные | `'FOO Bar'` → `'foo bar'` / `'FOO BAR'` |

### Поиск, замена и пробелы
```python
s.count('oo')          # Подсчёт подстрок
s.startswith('foo')    # Проверяет начало (True/False)
s.endswith('bar')      # Проверяет конец
s.find('foo')          # Возвращает индекс или -1, если нет
s.index('foo')         # Возвращает индекс или вызывает ошибку
s.replace('foo', 'grault', 2) # Замена (опционально: кол-во замен)
s.strip() / lstrip() / rstrip() # Удаление пробелов (всё/слева/справа)
```

Пример:
```python
raw = "first name;last name;email;address;"
fields = raw.split(";")    # Разделение строки на список по разделителю
for field in fields:
    print(field.strip())   # .strip() убирает лишние пробелы/переносы
```

### Классификация символов
- `isalnum()` – только буквы и цифры
- `isalpha()` – только буквы
- `isdigit()` – только цифры
- `isupper()` / `islower()` – все буквы верхнего/нижнего регистра
- `isspace()` – только пробелы

### Unicode
- `ord('A')` → `65` (порядковый номер символа)
- `chr(65)` → `'A'` (символ по номеру)
- Взаимозаменяемы: `chr(ord('A')) == 'A'`

## Условия и ветвление (Branching)
- `>=` больше или равен
- `!=` не равен
- **Транзитивность**: `==` транзитивна (`a==b` и `b==c` ⇒ `a==c`), `!=` – нетранзитивна.

```python
if guess == number:
    print('Угадал')
elif guess < number:
    print('Больше')
else:
    print('Меньше')
```

## Циклы
### Типы
- `for` – счетные циклы (counting loops)
- `while` – условные циклы (conditional loops)
- **Итерация** – однократное выполнение тела цикла.

### Блок `else` в циклах
- Если `else` присутствует, выполняется **только при штатном завершении** (без `break`).
- Если цикл прерван через `break`, `else` игнорируется.

### Управление потоком
- `break` – немедленный выход из цикла.
- `continue` – переход к следующей итерации, пропуск оставшегося кода.

```python
while running:
    # ... код ...
    if condition:
        break
else:
    print('Цикл завершен штатно.')

for i in range(1, 5): 
    print(i)
else:
    print('Цикл for завершен')
```

## Списки (Lists)
### Добавление и изменение
```python
words.append('python')    # Добавляет объект целиком
words.extend('python')    # Разбивает на символы и добавляет поэлементно
names.insert(0, 'Anders') # Вставка по индексу
```

### Удаление
```python
del numbers[1]            # По индексу (ключевое слово)
numbers.remove(1)         # По значению (удаляет первое вхождение)
item = names.pop(1)       # По индексу + возвращает удалённое значение
```

### Преобразования и поиск
```python
ip = '192.168.1.24'
numbers = ip.split('.')   # Строка → список (по разделителю)

words = ['Мы', 'учим', 'Python']
s = '*'.join(words)       # Список строк → строка

names.index('Roman')      # Индекс по значению
names.count(x)            # Количество вхождений
```

### Прочие методы
- `reverse()` – разворот списка на месте.
- `clear()` – очистка списка.
- `copy()` / `list()` / срезы `[:]` – создание копии.
- **Важно**: Строковые методы возвращают новые строки, списочные часто изменяют объект на месте.

### Списочные выражения (List Comprehensions)
```python
# Сокращённая запись вложенных циклов
numbers = [i * j for i in range(1, 5) for j in range(2)]
# Эквивалентно:
numbers = []
for i in range(1, 5):
    for j in range(2):
        numbers.append(i * j)
```

### Пример
```python
girls = ["Anna", "Maria", "Eva"]
girls.append("Brenda")           # Добавление в конец
girls.insert(1, "Kate")          # Вставка по индексу (положение, значение)
removed = girls.pop(0)           # Удаление и возврат элемента по индексу
copy_girls = girls.copy()        # Поверхностная копия списка

# Красивый вывод через запятую
cities = ['New York', 'San Francisco', 'Houston', 'Los Angeles']
print(*cities, sep=", ")
```

## Функции
### Параметры и аргументы
- **Формальные** – при описании функции.
- **Фактические** – при вызове.
- Значения по умолчанию: `def func(a, b=5, c=10):`
- `*args` – произвольное кол-во позиционных аргументов (кортеж).
- `**kwargs` – произвольное кол-во именованных аргументов (словарь).
- `return` – возврат значения.

### Область видимости
- `global x` – изменение глобальной переменной внутри функции.
- `nonlocal x` – изменение переменной во внешней (но не глобальной) области видимости.
- Локальные переменные не влияют на глобальные без `global`.

### Документация
```python
def printMax(x, y):
    '''Выводит максимальное из двух чисел.
    Оба значения должны быть целыми числами.'''
    # ...
print(printMax.__doc__)
```

### Пример
```python
for girl in girls:
    print(f"Name: {girl}")

# map() применяет функцию к каждому элементу последовательности
def double(x): return x * 2
numbers = [2, 4, 8]
result = list(map(double, numbers))  # [4, 8, 16]
```

## Практические примеры и задачи
### Угадывание числа (while + if)
```python
number = 23
running = True
while running:
    guess = int(input('Введите целое число:'))
    if guess == number:
        print('Поздравляем, Вы угадали!')
        running = False
    elif guess < number:
        print('Нет, загаданное число намного больше этого.')
    else:
        print('Нет, загаданное число намного меньше этого.')
else:
    print('Цикл while завершен.')
```

### Ввод скорости и классификация
```python
velocities = [0, 7.8, 11.2, 16.4]
results = ['error', 'упадет на Землю', 'станет спутником Земли', 'станет спутником Солнца', 'покинет Солнечную систему']
v = float(input())
if v in velocities: pass
else:
    velocities.append(v)
    velocities.sort()
print(results[velocities.index(v)])
```

### Поиск максимума и возврат значения
```python
def max(x, y):
    if x > y:
        return x
    elif x == y:
        return 'Числа равны'
    else:
        return y

```
