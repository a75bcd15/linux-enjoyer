# Номер порта
На одном [**IP-адресе**](ip.md) 65 535 портов. Каждый сервис занимает свой номер.

Условно порты разделяются на группы **системных** и **временных** портов. **Системные** (или стандартные, постоянные) от 1 до 1023, наиболее распространёнными из них являются:

Группа **временных** портов от 1024 до 65535 (в некоторых ОС эти пределы могут отличаться). Как правило, порты отправителя выбираются из доступного диапазона временных (т.е. динамически назначаемых) номеров портов.
- Обмен данными по протоколу [**TCP**](tcp.md) происходит следующим образом: номер порта отправителя произвольно выбирается для передачи данных получателю по известному порту. И как только будет отправлен первоначальный пакет, удаленное принимающее устройство свяжется с передающим устройством через установленные порты.
