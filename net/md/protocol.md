# Протокол
Современные сети состоят из разных систем, работающих на различных платформах. Для сообщения между системами в качестве общепонятного языка служит ряд **сетевых протоколов**. Все эти протоколы логически сгруппированы для совместной работы в **стек протоколов**. Протоколы действуют сходным образом с правилами употребления естественных языков и призваны решать следующие вопросы:
- Установление соединения;
- Согласование характеристик соединения;
- Форматирование данных;
- Обнаружение и исправление ошибок;
- Разрыв соединения.

## Порты протокола управления передачей (TCP) для серверных приложений
- **FTP (протокол передачи файлов) - порты 20 и 21**: широко используемый протокол для передачи файлов.
- [**SSH**](ssh.md) **(безопасная оболочка) - порт 22**: обеспечивает безопасную связь и удаленный доступ к устройствам по незащищенной сети.
- **Telnet - порт 23**: текстовый протокол, который позволяет взаимодействовать с удаленными устройствами по сетям.
- [**SMTP**](smtp.md) **(Simple Mail Transfer Protocol) - порт 25**: протокол для отправки и получения электронных писем.
- [**DNS**](dns.md) **(система доменных имен) - порт 53**: преобразует понятные человеку доменные имена в IP-адреса для облегчения связи между устройствами.
- [**HTTP**](http.md) **(протокол передачи гипертекста) - порт 80**: является основным протоколом, используемым в Интернете.
- **POP3 (протокол почтового отделения 3) - порт 110**: протокол для получения электронных писем с вашего почтового сервера.
- **IMAP (протокол доступа к интернет-сообщениям) - порт 143**: более продвинутый протокол электронной почты, который позволяет вам получать доступ к электронной почте и управлять ею на почтовом сервере.
- **HTTPS (безопасный протокол передачи гипертекста) - порт 443**: зашифрованная и безопасная версия HTTP.
- **RDP (протокол удаленного рабочего стола) - порт 3389**: разработанный Microsoft протокол для удаленного доступа к устройствам Windows.

## Порты протокола пользовательских дейтаграмм (UDP)
- [**DHCP**](dhcp.md) **(протокол динамической настройки хоста) - порты 67 и 68**: используется для распределения IP-адресов устройствам в сети.
- [**DNS**](dns.md) **(система доменных имен) - порт 53**: (та же функция, что и в TCP)
- **TFTP (Trivial File Transfer Protocol) - порт 69**: упрощенная версия FTP для быстрой и упрощенной передачи файлов.
- **SNMP (Simple Network Management Protocol) - порт 161**: позволяет отслеживать сетевые устройства и управлять ими, включая принтеры, маршрутизаторы и коммутаторы.
- **NTP (Network Time Protocol) - порт 123**: стандартный протокол, используемый для синхронизации времени между сетевыми устройствами.
