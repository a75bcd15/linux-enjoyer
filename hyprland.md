## Эстетичная загрузка

### Загрузка UEFI

Нужно скрыть все текстовые сообщения, которые выводит ядро Linux сразу после UEFI, чтобы был плавный переход к GRUB. В файле `/etc/default/grub` в строке `GRUB_CMDLINE_LINUX_DEFAULT` добавь параметры `quiet loglevel=3 vt.global_cursor_default=0`:
- `quiet` — отключает большинство информационных сообщений ядра при загрузке;
- `loglevel=3` — система будет показывать только критические ошибки, скрывая предупреждения и обычный текст;
- `vt.global_cursor_default=0` — отключает мигающий текстовый курсор в виртуальной консоли во время загрузки.

### Кастомизация GRUB

1. Скачать шаблон: `git clone https://github.com/uiriansan/LainGrubTheme.git`
2. Перейти в папку: `cd LainGrubTheme`
3. Перенести шаблон в папку: `sudo cp -r lain /boot/grub/themes/`
4. Открыть конфигурационный файл командой: `sudo nano /etc/default/grub`
5. Изменить строку, указав путь к файлу темы: `GRUB_THEME="/boot/grub/themes/lain/theme.txt"`
6. Обновить конфигурацию загрузчика: `sudo grub-mkconfig -o /boot/grub/grub.cfg`

**Примечания:**
- В интернете можно увидеть «анимированные» темы для GRUB. На самом деле эта анимация — это быстрая смена кадров из картинок, которая сильно нагружает загрузчик и может заставить его зависать. Лучше использовать красивый статичный фон с аккуратными иконками. 
- Можно вручную изменить название операционной системы при ситуации, когда имя отображается как `Arch Linux, with Linux linux`. После выполнения `grub-mkconfig` открой готовый конфиг `/boot/grub/grub.cfg`, найди строку, начинающуюся с `menuentry 'Arch Linux, with Linux linux'` и замени текст (например, `menuentry 'Arch Linux'`).
- Чтобы скрыть пункт "Advanced options", нужно в файле `/etc/default/grub` изменить строку на `GRUB_DISABLE_SUBMENU=y`.
- Скрыть пункт "UEFI Firmware Settings": `sudo chmod -x /etc/grub.d/30_uefi-firmware`

### Дисплейный менеджер Ly

Все настройки хранятся в файле `/etc/ly/config.ini`. 
``` bash
# Анимация в стиле матрицы
animation = matrix

# Делаем шлейф символов ярко-розовым (Hex: #FF69B4)
cmatrix_fg = 0x00FF69B4

# Делаем бегущую голову капли светло-розовой (Hex: #FFC0CB)
cmatrix_head_col = 0x00FFC0CB

# Розовый цвет текста (Hex: #F82D96)
fg = 0x00F82D96

# Цвет рамки ввода
border_fg = 0x00F82D96
```

Чтобы при входе в Hyprland экран не моргал черным, настроим ранний запуск графического драйвера (KMS):
   1. Открой файл конфигурации образов загрузки: `sudo nano /etc/mkinitcpio.conf`
   2. Найди строку `MODULES=()` и добавь туда драйвер amdgpu: `MODULES=(amdgpu)`
   3. Пересобери `initramfs`: `sudo mkinitcpio -P`

## Hyprland Экосистема

### Must-have пакеты

Список взят из официальной [вики](https://wiki.hypr.land/Useful-Utilities/Must-have/). Проверить, установлены ли они:
``` bash
for pkg in pipewire wireplumber xdg-desktop-portal-hyprland hyprpolkitagent qt5-wayland qt6-wayland noto-fonts dunst mako fnott swaync; do pacman -Q $pkg &>/dev/null && echo -e "\e[32m[Установлен]\e[0m $pkg" || echo -e "\e[31m[Отсутствует]\e[0m $pkg"; done
``` 

Среди демонов уведомлений `dunst`/`mako`/`fnott`/`swaync` достаточно иметь один, все четыре ставить не нужно. Устаноновить перечисленные пакеты:
``` bash
sudo pacman -S --needed pipewire wireplumber xdg-desktop-portal-hyprland hyprpolkitagent qt5-wayland qt6-wayland noto-fonts dunst
``` 

### Пользовательские приложения

1. `hyprpaper`
  - Легковесный установщик обоев рабочего стола. Если нужны сложные анимированные переходы, то `swww`.
2. `hyprpicker`
  - Пипетка для захвата цвета с экрана.
3. `hyprlauncher`
  - Минималистичное меню для поиска и запуска установленных приложений, альтернативы - `rofi-wayland` или `wofi`.
4. `hypridle`
  - Демон управления питанием и бездействием системы. Следит за тем, сколько времени не трогали мышь/клавиатуру, и может сначала приглушить яркость экрана, потом заблокировать систему, а затем отправить ПК в сон.
5. `hyprlock`
  - Быстрый экран блокировки, поддерживающий кастомизацию.
6. `xdg-desktop-portal-hyprland`
  - Отвечает за шеринг экрана (в Discord, Zoom, OBS) и системные диалоги выбора файлов.
7. `hyprsysteminfo`
  - Небольшая графическая утилита, отображающая информацию о системе. По сути — аналог `neofetch`/`fastfetch`, но с GUI.
8. `hyprsunset`
  - Утилита для регулировки цветовой температуры экрана («ночной режим» или blue light filter).
9. `hyprpolkitagent`
  - Официальный агент аутентификации (Polkit). Запрашивает пароль администратора (`root`), когда это требуется приложениям.
10. `hyprland-qt-support` и `hyprqt6engine`
  - Наборы инструментов и стилей для глубокой интеграции Qt5 и Qt6 приложений в графическую среду Hyprland.
11. `hyprpwcenter`
  - Графический интерфейс или апплет для управления PipeWire (звуком, микрофонами и аудио-потоками).
12. `hyprshutdown`
  - Простое графическое меню выхода из системы (выключить, перезагрузить, заблокировать, выйти).

**Дополнительно:**
- `hyprcursor`
  - Продвинутый формат тем курсора мыши для Wayland.
- `waybar`
  - Самая популярная и кастомизируемая статус-панель.
- `grim` + `slurp`
  - Классическая связка для создания скриншотов (`grim` делает снимок, `slurp` позволяет выбрать область мышкой).
- `cliphist`
  - Менеджер буфера обмена (чтобы сохранять историю скопированного текста и картинок).

Проверить, установлены ли пакеты:
``` bash
for pkg in swww hyprpaper hyprpicker hyprlauncher hypridle hyprlock xdg-desktop-portal-hyprland hyprsysteminfo hyprsunset hyprpolkitagent hyprland-qt-support hyprqt6engine hyprpwcenter hyprshutdown hyprcursor waybar grim slurp cliphist; do pacman -Q $pkg &>/dev/null && echo -e "\e[32m[Установлен]\e[0m $pkg" || echo -e "\e[31m[Отсутствует]\e[0m $pkg"; done
```

Установить пакеты:
``` bash
sudo pacman -S --needed swww hyprpaper hyprpicker hyprlauncher hypridle hyprlock xdg-desktop-portal-hyprland hyprsysteminfo hyprsunset hyprpolkitagent hyprland-qt-support hyprqt6engine hyprpwcenter hyprshutdown hyprcursor waybar grim slurp cliphist
``` 
### Плагины

1. `hy3`
  - Реализует ручной макет тайлинга. Позволяет создавать сложные вложенные контейнеры, например, нажатием клавиш можно сгруппировать три окна в одно с вкладками сверху.
2. `hyprscroller`
  - Превращает рабочее пространство в бесконечную горизонтальную ленту окон. Окна стоят в ряд, а ты скроллишь экран влево и вправо.
3. `hyprRiver` и `hyprNStack`
  - Реализуют альтернативные схемы тайлинга. `hyprRiver` копирует логику динамического композитора River, а `hyprNStack` расширяет стандартный Master-макет, позволяя делать не одну, а сколько угодно мастер-колонок (`N` штук).
4. `Hyprbars`
  - Возвращает классические заголовки окон с названием программы и кнопками «Закрыть» / «Свернуть».
5. `Borders++` 
  - Позволяет рендерить дополнительные рамки вокруг окон (двойные или тройные границы).
6. `Hyprtrails` 
  - Добавляет плавный, угасающий светящийся след (шлейф) за окном при его перемещении по экрану.
7. `Hyprglass`
  - Добавляет эффект «жидкого стекла» и искажения (преломления) света для окон с прозрачностью.
8. `Hyprexpo`
  - По нажатию клавиши все рабочие столы отдаляются и выстраиваются в сетку перед глазами.
9. `hyprspace`
  - Делает обзор рабочих столов в виде горизонтальной ленты с возможностью перетаскивать окна между ними мышкой.
10. `Hyprfocus` 
  - Создает визуальный эффект («вспышку», мерцание или кратковременное изменение размера) на окне, которое только что получило фокус.
11. `Hyprwinwrap`
- Позволяет превратить любое приложение (например, игру, видеоплеер 3D-сцену или браузер с живыми обоями) в фоновый рисунок рабочего стола.

Для установки плагина нужно воспользоваться встроенным менеджером:
  - Обновить базу: `hyprpm update`
  - Скачать официальный репозиторий: `hyprpm add https://github.com/hyprwm/hyprland-plugins` 
  - Включить плагин: `hyprpm enable hyprexpo`
  - Загрузить плагин в память: `hyprpm reload`
